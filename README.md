# 📚 Books-Store

## 🌐 Live URL

https://books-store-fawn.vercel.app

---

## 📖 Project Purpose

**Books-Store** is an online bookstore web application designed to provide users with a smooth and interactive experience for browsing, searching, and exploring books. The goal is to create a fast, responsive, and user-friendly platform for book lovers.

---

## ✨ Key Features

* 📚 Browse a collection of books
* 🔍 Search books by title or category
* 📱 Fully responsive design for all devices
* 🎞️ Interactive sliders and carousels
* ⚡ Fast performance and clean UI
* 🧭 Easy navigation between pages

---

## 📦 NPM Packages Used

* **react** – Frontend library for building UI
* **react-dom** – DOM rendering for React
* **react-router-dom** – Routing and navigation
* **swiper** – Modern touch slider for carousels
* **axios** – API data fetching
* **tailwindcss** *(if used)* – Utility-first CSS framework
